# Минисервис по отображению, созданию и редактированию публикаций

---

## Описание
разработать минисервис по отображению, созданию и редактированию публикаций. Требования:
- отображение статей: /articles
- создание статей: /articles/create
- редактирование статей: /articles/<id>/edit

---

##  Страницы

- **/articles**
  - отображение статей

- **/articles/create**
  - создание статей
 
- **/articles/<id>/edit**
  - редактирование статей
