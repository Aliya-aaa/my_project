# Минисервис по отображению, созданию и редактированию публикаций

Реализация платформы для публикации кулинарных рецептов на Python + Django с использованием PostgreSQL.

---

## Описание

Цель проекта — разработать веб-сайт, позволяющий пользователям просматривать и публиковать рецепты блюд.  
Сайт включает базовую аутентификацию и поиск по ингредиентам (дополнительное задание).

---

##  Страницы

- **/articles**
  - отображение статей

- **/articles/create**
  - создание статей
 
- **/articles/<id>/edit**
  - редактирование статей
